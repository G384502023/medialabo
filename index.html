    第1回WWW と HTML

    今回学習した内容は、HTMLの基本的な要素の構造とその使用方法についてだ。HTML要素は開始タグ、内容、終了タグで構成され、具体例として段落を示すpタグと強調を示すstrongタグなどがあった。また、改行を示すbrなどの空要素や、属性を使った要素の詳細な設定方法も学んだ。

  • 演習の成果:演習1-4:HTMLのさまざまな要素

    • 第2回 CSS による Web ページの装飾

    CSSはHTML要素を装飾する言語で、HTMLが先に記述され、その後にCSSで装飾を行う。例として、段落の文字色を緑にするには、HTMLのheadにリンク要素を追加し、style.cssファイルにp { color: green; }と記述する。複数要素の装飾はカンマで区切りる。箇条書きの点をなくすにはli { list-style-type: none; }を使用する。特定の要素を装飾するにはclass属性を用いるなどといった見た目を変えることを学んだ。

  • 演習の成果:演習2-3: 演習課題

    • 第3回 JavaScript の基本

    JavaScriptはWebページ上で複雑な機能を実現するプログラミング言語で、ページを開いた直後やボタンを押したときにプログラムが実行され、自動更新やアニメーションなどを可能にする。ブラウザAPIを利用することで、DOMの操作や現在位置の取得、2D・3Dグラフィックスの描画、リアルタイム通信が行うことができる。また、サードパーティAPIを使うことで、Twitterのツイート検索やGoogleマップの埋め込みなど、外部サービスとの連携も可能である。

  • 演習の成果:演習3-6: 演習課題

    • 第4回 DOM と JavaScript

    DOMは、Webページのコンテンツを階層化されたノードとして表現するデータ構造であり、JavaScriptを使って、このDOMを操作することで、ページの要素、属性、テキストにアクセスし、変更を加えたり新しい要素を追加したりすることが可能になる。例えば、documentノードを起点にしてHTML全体のノードを操作でき、これにより動的なWebページを実現することができる。
